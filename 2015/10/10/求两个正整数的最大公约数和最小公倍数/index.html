求两个正整数的最大公约数和最小公倍数

辗转相除法

辗转相除法， 又名欧几里德算法（Euclidean algorithm）,乃求两个正整数之最大公因子的算法。它是已知最古老的算法， 其可追溯至3000年前。

原理

javascript实现代码

function (m,n) {
    if (m<n) {
        temp=m; 
        m=n;
        n=temp;
    };
    do{
        temp=m%n;
        m=n;
        n=temp;
    }while(temp);
    var yue=m;              //最大公约数
    console.log(yue);
}

最小公倍数

最小公倍数的值可以通过最大公约数求得，即为输入的两个整数的乘积在除以最大公约数得到的值。
