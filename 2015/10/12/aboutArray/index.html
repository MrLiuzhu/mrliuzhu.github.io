部分数组函数的使用

题目

移除数组 arr 中的所有值与 item 相等的元素，请直接在给定的 arr 数组上进行操作

测试样例：
  removeWithoutCopy([1, 2, 2, 3, 4, 2, 2], 2)
返回：
  [1, 3, 4]

javascript代码

function removeWithoutCopy(arr, item) {
    var len=arr.length;
    for(var i=len-1;i>=0;i--){
        if(arr[i]==item)
            {
                arr.splice(i,1);
            }
    }
    return arr;
}

javascript代码2

function removeWithoutCopy(arr, item) {
    var len=arr.length;
    for(var i=0;i<len;i++){
        if(arr[i]==item)
            {
                arr.splice(i,1);
            }
        i--;
    }
    return arr;
}

splice函数说明

  array.splice(start,delCount,item1,item2….);//该函数从start位置开始删除delCount数量个元素，然后从start的位置插入后面的其他元素

start:开始删除元素的位置：包括这个位置；
delCount:删除元素的个数；
item1,item2…:删除后从start位置开始添加的元素
返回值：被删除项目组成的新数组。

例：var arr=[1,2,3,4];
arr.splice(1,2);//[1,4] 从位置为1的位置开始删除两个元素；
arr.splice(1,1,3);//[1,3,3,4] 从位置为1的地方删除一个元素，然后在这个位置添加元素3；

slice函数

  arrayObject.slice(start,end)//该函数从已有的数组中返回选定的元素

start:必须，开始选取元素的位置，如果是负数，那么它规定从数组尾部开始算起的位置。也就是说，-1 指最后一个元素，-2 指倒数第二个元素，以此类推。；
end:可选。规定从何处结束选取；
返回值：返回一个新的数组，包含从 start 到 end （不包括该元素）的 arrayObject 中的元素。

复制数组

  var newarr=arr.slice(0);

forEach函数

该函数不改变原函数的值

题目：统计数组arr中值等于item的元素出现的次数

function count(arr, item) {
    var num=0;
    arr.forEach(function(e){
           item==e?num++:num;
    })
    return num;
}

sort()函数

该函数默认已字符串的方式进行排序

返回值：对数组的引用。请注意，数组在原数组上进行排序，不生成副本。

查找数组重复元素

function duplicates(arr) {
    var a=arr.sort();
    var b=[];
    for(var i in a){
        if(a[i]==a[i-1] && b.indexOf(a[i])==-1) b.push(a[i]); 
    }
    return b;
}
