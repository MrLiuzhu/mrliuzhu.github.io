抛小球

题目

小东和三个朋友一起在楼上抛小球，他们站在楼房的不同层，假设小东站的楼层距离地面N米，球从他手里自由落下，每次落地后反跳回上次下落高度的一半，并以此类推知道全部落到地面不跳，求4个小球一共经过了多少米？(数字都为整数)

测试样例：
  100,90,80,70
返回：
  1020

c++代码

class Balls {
public:
    int calcDistance(int A, int B, int C, int D) {
        int sum=A+B+C+D;
        return 3*sum;
    }
};

分析

设每一个小球的初始高度为m,则每一个小球的第一次落地后，每次跳起至落地走过的路程为前一次的跳起高度，于是

Sn=m+m*(1+1/2+1/4+…+(1/2)^n)
Sn=m+m*((1-(1/2)^n)/(1-1/2))
当n趋近与无穷大时
Sn=3m
