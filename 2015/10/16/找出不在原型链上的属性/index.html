找出不在原型链上的属性

题目

找出对象 obj 不在原型链上的属性
1.返回数组，格式为 key: value
2.结果数组不要求顺序

输入例子

var C = function() {
        this.foo = 'bar'; 
        this.baz = 'bim';
    }; 
C.prototype.bop = 'bip'; 
iterate(new C());

输出例子

["foo: bar", "baz: bim"]

javascript代码

function iterate(obj) {
    var arr=[];
    for(var o in obj){
        if(obj.hasOwnProperty(o))
            {
                arr.push(o+": "+obj[o]);
            }
    }
    return arr;
}
