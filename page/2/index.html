被两个数整除的算法

题目

实现 fizzBuzz 函数，参数 num 与返回值的关系如下：
1、如果 num 能同时被 3 和 5 整除，返回字符串 fizzbuzz
2、如果 num 能被 3 整除，返回字符串 fizz
3、如果 num 能被 5 整除，返回字符串 buzz
4、如果参数为空或者不是 Number 类型，返回 false
5、其余情况，返回参数 num
输入例子：

fizzBuzz(15)

输出例子：

fizzbuzz

javascript代码

function fizzBuzz(num) {
    if(isNaN(num)){
        return false;
    }
    var str='';
    if(num%3===0){
        str+='fizz';
    }
    if(num%5===0){
        str+='buzz';
    }
    return str||num;
}

附：返回的是字符串，需要加引号

js笔记

本文主要记录一些常用js函数的用法

parseInt(string, radix)

参数说明：
string：必需。要被解析的字符串。
radix：可选。表示要解析的数字的基数。该值介于 2 ~ 36 之间。
如果省略该参数或其值为 0，则数字将以 10 为基础来解析。如果它以 “0x” 或 “0X” 开头，将以 16 为基数。如果该参数小于 2 或者大于 36，则 parseInt() 将返回 NaN。
提示：只有字符串中的第一个数字会被返回,开头和结尾的空格是允许的,如果字符串的第一个字符不能被转换为数字，那么 parseFloat() 会返回 NaN。
注意：如果 string 以 “0x” 开头，parseInt() 会把 string 的其余部分解析为十六进制的整数。如果 string 以 0 开头，那么 ECMAScript v3 允许 parseInt() 的一个实现把其后的字符解析为八进制或十六进制的数字。如果 string 以 1 ~ 9 的数字开头，parseInt() 将把它解析为十进制的整数。

parseInt("10");         //返回 10  
parseInt("19",10);      //返回 19 (10+9)  
parseInt("11",2);       //返回 3 (2+1)  
parseInt("17",8);       //返回 15 (8+7)  
parseInt("1f",16);      //返回 31 (16+15)  
parseInt("010");        //未定：返回 10 或 8

NumberObject.toString(radix)

作用：number转换成字符串
调用：NumberObject.toString(2);//NumberObject，2为转换成字符串的基数，此处将数字转换为2进制。
返回值：数字的字符串表示

部分数组函数的使用

题目

移除数组 arr 中的所有值与 item 相等的元素，请直接在给定的 arr 数组上进行操作

测试样例：
  removeWithoutCopy([1, 2, 2, 3, 4, 2, 2], 2)
返回：
  [1, 3, 4]

javascript代码

function removeWithoutCopy(arr, item) {
    var len=arr.length;
    for(var i=len-1;i>=0;i--){
        if(arr[i]==item)
            {
                arr.splice(i,1);
            }
    }
    return arr;
}

javascript代码2

function removeWithoutCopy(arr, item) {
    var len=arr.length;
    for(var i=0;i<len;i++){
        if(arr[i]==item)
            {
                arr.splice(i,1);
            }
        i--;
    }
    return arr;
}

splice函数说明

  array.splice(start,delCount,item1,item2….);//该函数从start位置开始删除delCount数量个元素，然后从start的位置插入后面的其他元素

start:开始删除元素的位置：包括这个位置；
delCount:删除元素的个数；
item1,item2…:删除后从start位置开始添加的元素
返回值：被删除项目组成的新数组。

例：var arr=[1,2,3,4];
arr.splice(1,2);//[1,4] 从位置为1的位置开始删除两个元素；
arr.splice(1,1,3);//[1,3,3,4] 从位置为1的地方删除一个元素，然后在这个位置添加元素3；

slice函数

  arrayObject.slice(start,end)//该函数从已有的数组中返回选定的元素

start:必须，开始选取元素的位置，如果是负数，那么它规定从数组尾部开始算起的位置。也就是说，-1 指最后一个元素，-2 指倒数第二个元素，以此类推。；
end:可选。规定从何处结束选取；
返回值：返回一个新的数组，包含从 start 到 end （不包括该元素）的 arrayObject 中的元素。

复制数组

  var newarr=arr.slice(0);

forEach函数

该函数不改变原函数的值

题目：统计数组arr中值等于item的元素出现的次数

function count(arr, item) {
    var num=0;
    arr.forEach(function(e){
           item==e?num++:num;
    })
    return num;
}

sort()函数

该函数默认已字符串的方式进行排序

返回值：对数组的引用。请注意，数组在原数组上进行排序，不生成副本。

查找数组重复元素

function duplicates(arr) {
    var a=arr.sort();
    var b=[];
    for(var i in a){
        if(a[i]==a[i-1] && b.indexOf(a[i])==-1) b.push(a[i]); 
    }
    return b;
}

抛小球

题目

小东和三个朋友一起在楼上抛小球，他们站在楼房的不同层，假设小东站的楼层距离地面N米，球从他手里自由落下，每次落地后反跳回上次下落高度的一半，并以此类推知道全部落到地面不跳，求4个小球一共经过了多少米？(数字都为整数)

测试样例：
  100,90,80,70
返回：
  1020

c++代码

class Balls {
public:
    int calcDistance(int A, int B, int C, int D) {
        int sum=A+B+C+D;
        return 3*sum;
    }
};

分析

设每一个小球的初始高度为m,则每一个小球的第一次落地后，每次跳起至落地走过的路程为前一次的跳起高度，于是

Sn=m+m*(1+1/2+1/4+…+(1/2)^n)
Sn=m+m*((1-(1/2)^n)/(1-1/2))
当n趋近与无穷大时
Sn=3m
