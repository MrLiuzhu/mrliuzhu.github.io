数组去重

题目

为 Array 对象添加一个去除重复项的方法

输入例子

[false, true, undefined, null, NaN, 0, 1, {}, {}, 'a', 'a', NaN].uniq()

输出例子

[false, true, undefined, null, NaN, 0, 1, {}, {}, 'a']

javascript代码

Array.prototype.uniq = function () {
    var hasNaN=false;
    var len=this.length;
    for(var i=0;i<len;i++){
        if(this[i]!==this[i]) hasNaN=true;    //NaN 与其他数值进行比较的结果总是不相等的，包括它自身在内
        for(var j=i+1;j<len;){
            if(this[j]===this[i] || (hasNaN&&(this[j]!==this[j]))){
                this.splice(j,1);            //此时当前位置变为下一个元素，因此不需要j++
                len-=1;
            }
            else{
                j++;
            }
        }
    }
    return this;
}

说明

NaN:NaN 与其他数值进行比较的结果总是不相等的，包括它自身在内,因此需要单独判断

求两个节点最近的父节点

题目

查找两个节点的最近的一个共同父节点，可以包括节点自身

输入描述

oNode1 和 oNode2 在同一文档中，且不会为相同的节点

javascript代码

function commonParentNode(oNode1, oNode2) {
    if(oNode1.contains(oNode2)){
       return oNode1;
       }
    else{
       return commonParentNode(oNode1.parentNode,oNode2);
    }
}

说明

nodeA.contains(nodeB);

这个方法起先用在 IE，用来确定nodeB是否包含在另一个nodeA中。如果包含，则返回true。

找出不在原型链上的属性

题目

找出对象 obj 不在原型链上的属性
1.返回数组，格式为 key: value
2.结果数组不要求顺序

输入例子

var C = function() {
        this.foo = 'bar'; 
        this.baz = 'bim';
    }; 
C.prototype.bop = 'bip'; 
iterate(new C());

输出例子

["foo: bar", "baz: bim"]

javascript代码

function iterate(obj) {
    var arr=[];
    for(var o in obj){
        if(obj.hasOwnProperty(o))
            {
                arr.push(o+": "+obj[o]);
            }
    }
    return arr;
}

二次封装函数

题目

实现函数 partialUsingArguments，调用之后满足如下条件：
1、返回一个函数 result
2、调用 result 之后，返回的结果与调用函数 fn 的结果一致
3、fn 的调用参数为 partialUsingArguments 的第一个参数之后的全部参数以及 result 的调用参数

输入例子

var a = 1; var b = 2; var c = 3; var d = 4;
var test = function (first, second, third, forth) {return first + second + third +forth;
partialUsingArguments(test, a, b)(c, d);

输出例子:

10

javascript

function partialUsingArguments(fn) {
    var newarr=Array.prototype.slice.call(arguments,1);
    var result=function(){
        return fn.apply(null,newarr.concat([].slice.call(arguments)));
    }
    return result;
}

说明

Array.prototype.slice.call(arguments,1); //将函数参数从第一位截取为数组
